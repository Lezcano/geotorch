trivialization(X, B)[source]¶

Parametrizes the manifold in terms of a tangent space :param X: A tensor, usually living in T_B M :type X: torch.nn.Tensor :param B: Point on M at whose tangent space we are trivializing :type B: torch.nn.Tensor

Returns

A tensor on the manifold

Return type

tensor (torch.nn.Tensor)

Note

This function should be surjective, otherwise not all the manifold will be explored
